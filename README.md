# [Sifreli-Gonderici]

Python ile geliştirilmiş, **AES-256** şifreleme algoritmasını kullanarak dosya ve metin mesajlarını güvenli bir şekilde şifreleyen ve gönderime hazır hale getiren basit bir araçtır.

## ✨ Özellikler

* **AES-256 Şifreleme:** Verileriniz endüstri standardı olan AES-256 ile korunur.
* **Dosya Şifreleme:** Her türlü dosya formatını (.jpg, .pdf, .docx vb.) şifreleyebilir.
* **Basit Kullanım:** Karmaşık ayarlar gerektirmeyen, doğrudan işlev odaklı yapı.
